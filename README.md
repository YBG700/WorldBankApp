
# Banking Application (C# .NET MAUI)

A cross-platform banking application built with C# and .NET MAUI that allows users to create and manage bank accounts with ease. The app supports account creation, user information input, account type selection, deposits, withdrawals, and special offers for student accounts.

---

## Features

- **Create a New Account**  
  Users can create a new bank account by providing personal details.

- **User Information Management**  
  Enter and update name, age, email, phone number, and date of birth.

- **Account Type Selection**  
  Choose between Chequing, Savings, or Everyday accounts.

- **Transactions**  
  Make deposits and withdrawals securely.

- **Student Accounts**  
  Special accounts for students that receive exclusive deals and cashback offers.

---

## Technologies Used

- C#  
- .NET MAUI (Multi-platform App UI)  
- MVVM Pattern (recommended for architecture)  
- SQLite or other local database (for persistent storage)  
- .NET Libraries for data handling and UI

### Run the Application

---

## Usage

1. **Create an Account**  
   - Enter your personal information: Name, Age, Email, Phone Number, Date of Birth.  
   - Select the account type: Chequing, Savings, or Everyday.  
   - Indicate if you are a student for special benefits.

2. **Manage Funds**  
   - Deposit funds into your account.  
   - Withdraw funds with validation to prevent overdraft.

3. **Student Benefits**  
   - If your account is marked as a student account, you will receive exclusive cashback offers and deals displayed within the app.
